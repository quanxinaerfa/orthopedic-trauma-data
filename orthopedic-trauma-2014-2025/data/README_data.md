## Real Data Access

The de-identified patient-level dataset analysed in the manuscript contains
protected health information and **cannot be shared publicly** under the terms
of the ethics approval (Haimen District People's Hospital Ethics Committee,
ref: 2026-KY015-01).

**To request access for legitimate research purposes:**

1. Email the corresponding author: `hmrysci@163.com`
2. Provide:
   - A formal data access request with study protocol
   - IRB approval from your home institution
   - A signed data use agreement

Approval typically takes 4–6 weeks.

## Synthetic Sample

`synthetic_sample_n100.csv` is a fabricated dataset of 100 rows with **identical
column structure** to the real data. It is generated by randomly perturbing
real values with no possibility of re-identification, and **must not be used for
substantive analysis**. Its sole purpose is to verify that the analysis pipeline
runs without error.

To regenerate the synthetic sample, see `R/generate_synthetic_sample.R`.

## File Format Specification

- **Encoding:** UTF-8 (Chinese characters preserved)
- **Delimiter:** comma (`,`)
- **Date format:** ISO 8601 (`YYYY-MM-DD`)
- **Missing values:** empty string or `NA`
- **Currency:** RMB (Chinese Yuan), no thousands separator
